# 📦 Product Scraper Dashboard

This is a full-stack mini project that scrapes product data from a website, stores it, and displays it in a simple dashboard.

---

## 🚀 Features

- Web scraping using Node.js
- Automatic data update using CRON job
- REST API using Express.js
- Frontend dashboard to display products
- Simple and responsive UI

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- Axios
- Cheerio
- HTML, CSS, JavaScript

---

## 📂 Project Structure
scraper/
api/
frontend/

---

## ⚙️ How to Run

### 1. Install dependencies
npm install

### 2. Run scraper

### 3. Run API

### 4. Open frontend
Open `index.html` in browser

---

## 📊 Output

- Scrapes 10–20 products
- Shows them in dashboard via API
